# 统一数据管理框架错误码

>
> 以下仅介绍本模块特有错误码，通用错误码请参考[通用错误码说明文档](../errorcode-universal.md)。

##  20400002 存储的数据不支持解析

**错误信息**

The stored data does not support parsing.

**错误描述**

存储的数据不支持解析。

**可能原因**

1. 存储的非自定义数据类型与对应的数据结构不匹配。
3. 错误地使用接口。

**处理步骤**

1. 开发者排查存储的非自定义数据类型与对应的数据结构不匹配。
2. 开发者排查是否按接口文档正确使用接口。

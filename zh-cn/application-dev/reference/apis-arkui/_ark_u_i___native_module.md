# ArkUI_NativeModule


## 概述

提供ArkUI在Native侧的UI能力，如UI组件创建销毁、树节点操作，属性设置，事件监听等。

提供ArkUI在Native侧的注册手势回调的能力。

提供ArkUI在Native侧动画回调的能力。

**起始版本：** 12

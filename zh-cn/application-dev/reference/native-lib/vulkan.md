# Vulkan

[Vulkan](https://www.vulkan.org/)是一套2D和3D绘图应用程序接口，具有跨平台、高性能的特点。OpenHarmony现已支持Vulkan v1.3.231版本。

## 支持的标准能力

支持Vulkan v1.3.231版本的部分接口能力，具体请见[支持的Vulkan接口列表](vulkan-symbol.md)。

## 支持的扩展能力

提供OpenHarmony平台扩展的Vulkan能力，扩展了使用OHNativeWindow创建Vulkan Surface的能力，以及获取OH_NativeBuffer和OH_NativeBuffer属性的能力。

Vulkan扩展能力具体请见[Vulkan](_vulkan.md)。

针对Vulkan的开发使用具体请见[Vulkan开发指导](vulkan-guidelines.md)。

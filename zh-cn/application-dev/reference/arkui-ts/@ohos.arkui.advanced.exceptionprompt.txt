

# @ohos.arkui.advanced.ExceptionPrompt（异常提示控件）

ExceptionPrompt适用于有离线内容可显示的情况。

**说明：**

该组件从API Version 11开始支持。后续版本如有新增内容，则采用上角标单独标记该内容的起始版本。

## 导入模块

```ts
import {ExceptionPrompt} from '@ohos.arkui.advanced.exceptionprompt';
```

##  子组件

无

##  Component

**装饰器类型：**@Component

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## 参数

| 名称        | 类型 | 装饰器类型 | 必填        | 说明                            |
| ----------- | ---------- | ------| --------------------------------- | --------------------------------- |
| Type | Number | @Prop | 是   | 指定当前ExceptionPrompt的类型。1、2、3、4对应四种显示状态 |
| Icon | String | @Prop | 否    | 指定当前ExceptionPrompt的异常图标式样。 |
| TipContent | String | @Prop | 否 | 指定当前ExceptionPrompt的文字提示式样。 |
| ContentText | String | @Prop | 否 | 指定当前ExceptionPrompt有网但是获取不到内容XX，XX包含但不限于“信息”，“资料”，“图片”等。 |
| HardwareStatus | boolean | @Prop | 否 | 指定当前硬件打开状态。默认true：打开状态或者false。 |
| ThemeColorStatus | boolean | @Prop | 否 | 指定当前主题背景是浅色还是深色，true：默认浅色背景，false：深色背景。 |

## Type

Type指定ExceptionPrompt的类型。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

| 类型 |           说明           |
| :--: | :----------------------: |
|  1   |      网络未连接状态      |
|  2   |    网络连接不稳定状态    |
|  3   |     连不上服务器状态     |
|  4   | 有网但是获取不带内容状态 |

## 事件

| 名称                                                   | 功能描述                                 |
| ------------------------------------------------------ | ---------------------------------------- |
| ReconnectionFunction: () => void = () => { }           | 点击左侧文本，变为正在连接状态事件       |
| ConfigureNetworkFunction: () =&gt; void = () =&gt; { } | 点击设置网络跳转到设置网络网络弹出框事件 |

## 示例 2

```ts
import { ExceptionPrompt } from '../../../../../ExceptionPrompt/src/main/ets/components/mainpage/ExceptionPrompt'
const HEIGHT = 35
@Entry
@Component
struct Index {
  @State HardwareStatus: boolean = true //硬件开关
  @State ThemeColorStatus: boolean = true //背景主题
  @State TipContent: string = '' //异常提示内容
  @State ContentText: string = ''
  @State Type: number = 0 //异常提示状态切换
  @State ButtomText: string = ''
  @State MAP_HEIGHT: string = '30%' //背景图片高
  @State HEIGHT: number = 30
  controller: TextInputController = new TextInputController()

  build() {
    Column() {
      ExceptionPrompt(
        {
          Type: this.Type,
          Icon: '',
          TipContent: this.TipContent,
          HardwareStatus: this.HardwareStatus,
          ContentText: this.ContentText,
          ThemeColorStatus: this.ThemeColorStatus,
          ReconnectionFunction: () => {
            animateTo({ duration: 800 }, () => {
              this.ButtomText = '正在连接'
            })
          },
          ConfigureNetworkFunction: () => {
            animateTo({ duration: 1500 }, () => {
              this.ButtomText = '无网络情况，请设置网络'
            })
          },
        }
      )
```

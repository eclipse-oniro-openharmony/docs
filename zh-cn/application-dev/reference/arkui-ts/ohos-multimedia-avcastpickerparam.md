# @ohos.multimedia.avCastPickerParam (投播参数)

本模块提供创建投播组件AVCastPicker使用的相关参数，需和投播组件[AVCastPicker](ohos-multimedia-avcastpicker.md)配合使用。

> **说明：**
>
> - 本模块首批接口从API version 11开始支持。后续版本的新增接口，采用上角标单独标记接口的起始版本。
> - 示例效果请以真机为准，当前IDE预览器无实际投播功能。

## AVCastPickerState

枚举，表示投播组件的状态。

**系统能力：** SystemCapability.Multimedia.AVSession.AVCast

| 名称 | 描述 |
| ----- | -------- |
| STATE_APPEARING | 投播组件开始展示。 |
| STATE_DISAPPEARING | 投播组件完成展示。 |

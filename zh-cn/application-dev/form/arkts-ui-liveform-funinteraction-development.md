# 趣味交互类型互动卡片开发指导

本文档提供了趣味交互类型互动卡片的开发指导，包括：卡片配置开发和游戏内容具体开发。

## 开发接入指导

趣味交互类型互动卡片当前仅支持基于鸿蒙快游戏引擎开发的卡片小游戏。卡片主要包含三个状态：激活态、暂停态和非激活态。分别对应游戏进行、游戏暂停、游戏结束。

游戏内容具体开发可参考[互动卡片小游戏开发指导](https://developer.huawei.com/consumer/cn/doc/quickApp-Guides/quickgame-interact-card-dev-0000002045919412)。

端侧卡片配置开发可参考[趣味交互卡片配置指导](arkts-ui-widget-configuration.md#funinteractionparams标签)。

# 元能力子系统Changelog

## cl.formkit.1 卡片可加载module、api安全校验加固

**访问级别**

其他

**变更原因**

卡片可加载module、api安全校验加固。

**变更影响**

此变更不涉及应用适配。


变更前：卡片代码内存在绕过安全校验使用未携带@form标签的接口。


变更后：运行时校验加固，只允许使用白名单允许使用的接口。

**起始 API Level**

API 18

**变更发生版本**

从OpenHarmony SDK 5.1.0.56开始。

**变更的接口/组件**

不涉及接口

**适配指导**

行为变更：默认行为变更，无需适配。

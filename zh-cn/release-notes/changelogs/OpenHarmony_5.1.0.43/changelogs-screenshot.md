## cl.screenshot.1 pick截屏时跳过隐私窗口改为隐私窗口显示全黑。

**访问级别**

公开接口

**变更原因**

截屏时跳过隐私窗口会显示被隐私窗口遮挡的内容，有隐私风险。

**变更影响**

该变更为不兼容变更。


变更前：

截屏时跳过隐私窗口。


变更后：

截屏时隐私窗口显示全黑。


**起始API Level**

API 12

**变更发生版本**

从OpenHarmony SDK 5.1.0.43开始。

**变更的接口/组件**

pick

**适配指导**

默认行为变更，无需适配，但应注意变更后的行为是否对整体应用逻辑产生影响。

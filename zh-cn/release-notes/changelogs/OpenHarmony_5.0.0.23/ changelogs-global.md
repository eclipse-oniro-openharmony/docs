# Global变更说明

## cl.global.1 ohos.i18n部分接口入参类型校验失败错误码抛出变更

**变更原因**

401错误码是通用的参数类型检查错误码，原890001错误码不再在参数类型检查场景抛出

**变更影响**

该变更为非兼容性变更。
更改后应用开发者需要捕捉新的错误码进行错误处理。
当前IDE已有编译阶段参数类型检查，如果使用IDE进行编译，基本无影响。

**起始API Level**

12

**变更发生版本**

从OpenHarmony SDK 5.0.0.23 版本开始。
